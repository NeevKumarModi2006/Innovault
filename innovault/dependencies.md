# Project Dependencies

Since `npm` was not available during setup, please install the following dependencies manually when you have access to a terminal with Node.js installed.

## Frontend (`./innovault/frontend`)

Navigate to `innovault/frontend` and run:

```bash
npm install
npm install react react-dom react-router-dom framer-motion lucide-react axios
npm install -D vite @vitejs/plugin-react tailwindcss postcss autoprefixer
```

## Backend (`./innovault/backend`)

Navigate to `innovault/backend` and run:

```bash
npm install
npm install express mongoose dotenv cors jsonwebtoken bcryptjs multer
npm install -D nodemon
```
